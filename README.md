# yeah-boot-admin

`yeah-boot-admin` 是一个准备按高规格标准持续建设的自研后台脚手架项目。

项目最初参考过若依体系的前端思路，但后续目标已经调整为：沉淀一套更易维护、更适合长期演进、更适合开源协作的前后端脚手架基础设施。

## 当前定位

- 自研后台管理脚手架
- 面向长期维护和二次开发
- 以工程质量、规范一致性、文档完整性为第一优先级
- 目标是可开源，而不是一次性的业务模板

## 当前状态

仓库目前仍处于前端后台原型阶段，部分页面和结构已存在，但工程规范、基础设施抽象和文档体系仍在建设中。

后续开发将优先推进：

- 架构收敛
- 目录规范
- 路由与权限设计
- 请求层与状态管理设计
- 通用后台组件抽象
- 项目文档与开发日志制度化

## 开发环境

### Node 版本

- `^20.19.0 || >=22.12.0`

### 安装依赖

```sh
npm install
```

### 本地开发

```sh
npm run dev
```

### 类型检查与构建

```sh
npm run build
```

### 代码检查

```sh
npm run lint
```
